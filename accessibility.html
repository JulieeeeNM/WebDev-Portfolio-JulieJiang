Accessibility Statement

I am committed to ensuring digital accessibility for people with disabilities. I continually improve the user experience for everyone by applying the relevant accessibility standards.

Implemented Accessibility Features

Semantic HTML

Proper use of heading hierarchy, landmarks, and ARIA attributes to ensure screen reader compatibility.

Keyboard Navigation

All interactive elements are accessible via keyboard, with visible focus states.

Color Contrast

Text and interactive elements meet WCAG 2.1 AA contrast ratio requirements.

Alternative Text

All meaningful images include descriptive alt text for screen reader users.

Responsive Design

The site is fully responsive and usable on all device sizes.

Reduced Motion

Animation preferences are respected based on user system settings.

Testing & Compliance

This website has been tested using the following methods:

  • Automated testing with WAVE and axe tools
  • Manual keyboard navigation testing
  • Screen reader testing with NVDA and VoiceOver
  • Color contrast verification
  • User testing with people with disabilities

I aim to meet WCAG 2.1 Level AA standards, though some third-party content may not be fully accessible.
